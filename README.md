
# StellarProof

> Reusable identity infrastructure for the Stellar ecosystem.

**Live:** [stellarproof.org](https://stellarproof.org)

---

## What is StellarProof?

StellarProof is a shared identity layer that lets users verify their identity
once and reuse that verification across all Stellar anchors — without
re-submitting documents, re-uploading selfies, or repeating KYC flows.

The Stellar ecosystem has hundreds of anchors. Every one runs independent KYC.
Every user starts from scratch. Every time.
